LerretFixtureMono — license

LerretFixtureMono.woff2 was authored from scratch for the Lerret test fixture.
It copies no existing typeface and carries no third-party copyright. Every
glyph is a solid full-em rectangle — a deliberately distinctive shape so the
fixture can prove, by eye and by DOM check, that a custom font dropped into
`.lerret/_fonts/` is auto-registered and applied.

This file is NOT a font file. It sits beside the font in `_fonts/` and the
Lerret loader skips it: only WOFF/WOFF2/TTF/OTF files are registered, so an
unsupported file in `_fonts/` is ignored without breaking the valid fonts.

# A folder is a canvas

Every card here is a real file in this folder — `.jsx` components and `.md`
notes (like this one). Edit a file, and the canvas re-renders instantly.

The framed, tinted sections below are **groups** — real sub-folders that keep
related designs together:

- **brand** — tokens + identity, as reusable data-driven design
- **social** — one brand, every post format
- **live** — designs that update themselves
- **launch** — ready-to-ship marketing
- **slides** — a whole talk deck, in code
- **site** — web assets (OG card, /now banner, favicon)

Every group has an `about.md` explaining it. Zoom out to take it all in, or open
any file to tinker.

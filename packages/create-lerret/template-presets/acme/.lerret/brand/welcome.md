# The Acme brand kit

Every card on this page renders straight from a React component in
`.lerret/brand/`. Edit the file, hit save, and the artboard updates in place —
no rebuild, no refresh.

## What this page shows

- **Business card** declares a `propsSchema`, so the studio gives it a typed
  **data editor** and shows a **⚠️ validation badge** when a required prop
  (here, `name`) is missing.
- The two business-card artboards are **variants** of one file, fed by
  `business-card.data.json`.
- This group sets a warm `presentation.background` in its `config.json`, which
  is why the page sits on linen rather than the default canvas color.

Right-click any artboard — or this page — to see the same actions the `⋮` menu
offers.

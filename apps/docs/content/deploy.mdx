# Deployment

This page is a stub. Topics that will land here:

- **Local CLI mode** — `lerret dev` against a local folder. Vite HMR. The dev-default.
- **Hosted mode** — the studio served as a static site; users open a local folder via the File System Access API. Chromium-only.
- **Self-host** — the same hosted-mode static build, packaged for self-deployment.
- **Trust acknowledgement** — first-time per-folder confirmation before user code runs.
- **The `.lerret/.state/` sidecar** — local studio state, gitignored automatically.

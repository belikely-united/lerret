# Examples

The assets indie makers and founders actually ship — Product Hunt launches, feature drops, stars-milestone celebrations, testimonials, release graphics, hiring posts, talk promos, year-in-review recaps. Each is a complete Lerret asset: drop the file into your `.lerret/` directory and the artboard appears on the canvas. Save, render, post.

> **The previews on this page were rendered with Lerret.** Every image below is the actual output of `@lerret/cli export` run against the source code shown next to it — no Photoshop, no mockups. The docs site uses the product to make its own marketing.

> **Where to put the files.** Inside your project's `.lerret/` folder, put each `.jsx` under a page folder (e.g. `.lerret/marketing/product-hunt.jsx`). Co-located data files live alongside (`.lerret/marketing/product-hunt.data.json`). See [Concepts → folder-canvas model](/concepts/#the-folder-canvas-model).

## 1. Product Hunt launch day

![Product Hunt launch card — dark warm gradient, "LIVE ON PRODUCT HUNT" pill, "#1 Product of the Day", giant product name, faux upvote counter and producthunt.com CTA](/examples/product-hunt.jpg)

## 2. Feature launch announcement

**Use case.** The "Introducing X" Twitter/LinkedIn post. Bold gradient text, NEW pill, version eyebrow, supporting description, primary CTA. 1200×675 — fits in-feed at full width.

![Feature launch announcement — near-black background with indigo-pink diagonal gradient, "NEW" pill + "NEW IN v2.0" mono pill, huge "Real-time multiplayer." gradient text, description, white CTA button](/examples/feature-launch.jpg)

## 3. GitHub stars milestone

**Use case.** "We just hit X stars" celebrations are catnip on dev Twitter. This one uses a giant gold-gradient number on a cosmic background with star dots.

![Stars milestone card — deep navy-purple cosmic gradient, scattered white star dots with subtle glow, "MILESTONE" eyebrow, giant gold-gradient "10,000" number, "stars on GitHub" label, repo URL and "Thank you, everyone." line](/examples/stars-milestone.jpg)

**Path.** `.lerret/marketing/stars-milestone.jsx`

## 5. Release / version announcement

**Use case.** When you ship a major version, post a card with the version number, codename, headline, and 3–4 highlight bullets. Sized 1200×800 — tall enough to fit the bullet list without crowding.

![Release card — deep blue cosmic gradient with subtle grid lines, "v2.0" pill chip + "CODENAME · HYDRA" mono label, large "Multiplayer, scopes, and a new API." headline, four numbered feature bullets in cream accent](/examples/release-card.jpg)

**Path.** `.lerret/marketing/release-card.jsx`

**Notes.** The grid pattern uses **two layered linear-gradients** as the `background-image` plus a **radial `mask-image`** so the grid fades out at the edges — pure CSS, no SVG. That trick translates very well across captures.

```jsx
              borderRadius: 100, fontSize: 15, fontWeight: 700,
            }}>{p}</div>
          ))}
        </div>
        <div style={{ fontSize: 16, fontWeight: 700, opacity: 0.75, fontFamily: 'ui-monospace, Menlo, monospace' }}>
          {url} →
        </div>
      </div>
    </div>
  );
}
```

**Notes.** Two soft gradient orbs (one cream-white in the bottom-left, one coral in the top-right) give the card depth without darkening. The "NOW HIRING" pill is inverted dark on warm — strong contrast that earns attention. Perks are loose pills so swapping them is one prop change.

## 8. Year in review (bento)

**Use case.** Annual recap with key metrics. Vertical 1200×1260 — Instagram-story friendly and great for LinkedIn long-scroll feeds. Bento grid of four stats, each with its own brand color.

![Year in review card — deep black canvas, "A YEAR AT BELIKELY UNITED" eyebrow, huge gradient "2026" number, "In numbers." subtitle, 2x2 bento grid of four stats: 127,000 users onboarded (cyan), $2.4M ARR (gold), 12 releases shipped (purple), 847 GitHub stars (green) — each in its own colored card](/examples/year-review.jpg)

**Path.** `.lerret/marketing/year-review.jsx`

```jsx
export const meta = {
  dimensions: { width: 1200, height: 1260 },
  label: 'Year in review (bento)',
  tags: ['recap', 'stats', 'year-review'],
  propsSchema: {
    year: { type: 'string', default: '2026' },
    company: { type: 'string', default: 'Belikely United' },
    stat1Value: { type: 'string', default: '127,000' },
    stat1Label: { type: 'string', default: 'users onboarded' },
    stat2Value: { type: 'string', default: '$2.4M' },
    stat2Label: { type: 'string', default: 'ARR' },
    stat3Value: { type: 'string', default: '12' },
    stat3Label: { type: 'string', default: 'releases shipped' },
    stat4Value: { type: 'string', default: '847' },
    stat4Label: { type: 'string', default: 'GitHub stars' },
  },
};

export default function YearReview(props) {
  const stats = [
    { v: props.stat1Value, l: props.stat1Label, bg: '#1B2A3B', accent: '#22D3EE' },
    { v: props.stat2Value, l: props.stat2Label, bg: '#2D1810', accent: '#FFC04A' },
    { v: props.stat3Value, l: props.stat3Label, bg: '#2A1B3A', accent: '#A78BFA' },
    { v: props.stat4Value, l: props.stat4Label, bg: '#0F2A1B', accent: '#34D399' },
  ];
  return (
    <div style={{
      width: '100%', height: '100%', boxSizing: 'border-box',
      padding: '80px', background: '#0A0A0A', color: '#FFF',
      fontFamily: 'system-ui, -apple-system, sans-serif',
      display: 'flex', flexDirection: 'column', gap: 56,
    }}>
      <div>
        <div style={{
          fontSize: 14, letterSpacing: '0.35em', opacity: 0.55,
          fontFamily: 'ui-monospace, Menlo, monospace', fontWeight: 700,
        }}>A YEAR AT {props.company.toUpperCase()}</div>
        <div style={{
          fontSize: 168, fontWeight: 900, lineHeight: 0.88,
          letterSpacing: '-0.05em', marginTop: 18,
          fontVariantNumeric: 'tabular-nums',
          background: 'linear-gradient(135deg, #FFF 0%, #6E6E6E 100%)',
          WebkitBackgroundClip: 'text', WebkitTextFillColor: 'transparent',
          backgroundClip: 'text',
        }}>{props.year}</div>
        <div style={{ fontSize: 24, opacity: 0.6, marginTop: 16, fontWeight: 500 }}>
          In numbers.
        </div>
      </div>

      <div style={{
        flex: 1, display: 'grid',
        gridTemplateColumns: '1fr 1fr', gridTemplateRows: '1fr 1fr', gap: 20,
      }}>
        {stats.map((s, i) => (
          <div key={i} style={{
            padding: 44, borderRadius: 28,
            background: s.bg, color: '#FFF',
            position: 'relative', overflow: 'hidden',
            display: 'flex', flexDirection: 'column', justifyContent: 'space-between',
          }}>
            <div style={{
              position: 'absolute', bottom: -100, right: -100,
              width: 280, height: 280, borderRadius: '50%',
              background: `radial-gradient(circle, ${s.accent}40, transparent 60%)`,
              filter: 'blur(8px)',
            }} />
            <div style={{
              fontSize: 13, letterSpacing: '0.32em', opacity: 0.55,
              fontFamily: 'ui-monospace, Menlo, monospace', fontWeight: 700,
              position: 'relative',
            }}>0{i + 1}</div>
            <div style={{ position: 'relative' }}>
              <div style={{
                fontSize: 92, fontWeight: 900, lineHeight: 0.95,
                letterSpacing: '-0.04em', fontVariantNumeric: 'tabular-nums',
                color: s.accent,
              }}>{s.v}</div>
              <div style={{
                fontSize: 19, marginTop: 10, opacity: 0.85, fontWeight: 500,
              }}>{s.l}</div>
            </div>
          </div>
        ))}
      </div>
    </div>
  );
}
```

**Notes.** Each bento cell has a **dark tinted background** (just slightly off from pure black) and a **soft-glow orb** in its accent color sitting in the corner. That combination — pure black canvas + tinted cells + colored glows — is the modern "bento" recipe (Apple, Linear, Stripe Sigma all use variants). Different accent per cell gives variety without needing a unifying palette.

# Examples

The assets indie makers actually publish — but treated as **posters**, not UI banners. Each one leans on art-direction conventions you'd recognise from print: editorial mastheads, tour-poster typography, magazine-cover quote treatments, Swiss modernist grids, brutalist hand-set type. Plain React; no design tool round-trip.

> **The previews on this page were rendered with Lerret.** Every image below is the output of `@lerret/cli export` against the source code shown next to it.

> **Where to put the files.** Inside your project's `.lerret/` folder, put each `.jsx` under a page folder (e.g. `.lerret/marketing/product-hunt.jsx`). Co-located data files live alongside (`.lerret/marketing/product-hunt.data.json`).

## 1. Product Hunt launch — tour poster

**Use case.** The card you ship the morning you launch. Treats the moment like a printed handbill: serif italic product name as the centerpiece, condensed sans callout in PH orange, a hairline frame, tour-style date strip. Portrait 1200×1500, Instagram-grid and LinkedIn friendly.

![Product Hunt launch poster — cream background with hairline border, monospace masthead "The Belikely Press · NO.001 · Launch Day Edition", condensed orange "WE'RE / LIVE / ON" stacked, massive italic serif product name, italic tagline, date strip with sienna producthunt.com](/examples/product-hunt.jpg)

**Path.** `.lerret/marketing/product-hunt.jsx`

```jsx
export const meta = {
  dimensions: { width: 1200, height: 1500 },
  label: 'Product Hunt — launch poster',
  tags: ['poster', 'product-hunt', 'launch'],
  propsSchema: {
    productName: { type: 'string', default: 'Lerret', required: true },
    tagline: { type: 'string', default: 'A folder of React components, rendered as a visual canvas.' },
    launchDate: { type: 'string', default: '11.22.26' },
    issue: { type: 'string', default: 'NO. 001' },
  },
};

export default function ProductHuntPoster({ productName, tagline, launchDate, issue }) {
  return (
    <div style={{
      width: '100%', height: '100%', boxSizing: 'border-box',
      background: '#F5EBD8', color: '#1A1410',
      fontFamily: 'Georgia, "Times New Roman", serif',
      position: 'relative', overflow: 'hidden', padding: '72px 80px',
      display: 'flex', flexDirection: 'column', justifyContent: 'space-between',
    }}>
      {/* Hairline frame */}
      <div style={{
        position: 'absolute', inset: '32px 40px',
        border: '1px solid rgba(26,20,16,0.25)', pointerEvents: 'none',
      }} />

      {/* Masthead */}
      <div style={{
        position: 'relative', display: 'flex', justifyContent: 'space-between',
        fontFamily: 'ui-monospace, Menlo, monospace',
        fontSize: 14, letterSpacing: '0.25em', textTransform: 'uppercase',
      }}>
        <span>The Belikely Press · {issue}</span>
        <span>Launch Day Edition</span>
      </div>

      {/* Stacked callout + product name */}
      <div style={{ position: 'relative' }}>
        <div style={{
          fontFamily: 'system-ui, -apple-system, sans-serif',
          fontSize: 88, fontWeight: 900, letterSpacing: '-0.04em',
          lineHeight: 0.86, textTransform: 'uppercase', color: '#DA552F',
        }}>
          We're<br />Live<br />on
        </div>
        <div style={{
          fontSize: 220, fontWeight: 400, letterSpacing: '-0.05em',
          lineHeight: 0.9, fontStyle: 'italic', marginTop: 8,
        }}>
          {productName}.
        </div>
      </div>

      {/* Tagline */}
      <div style={{ position: 'relative', maxWidth: '78%' }}>
        <div style={{ fontSize: 30, lineHeight: 1.3, fontStyle: 'italic' }}>
          {tagline}
        </div>
      </div>

      {/* Tour-date strip */}
      <div style={{
        position: 'relative', borderTop: '2px solid #1A1410',
        paddingTop: 28, display: 'flex',
        justifyContent: 'space-between', alignItems: 'baseline',
      }}>
        <div style={{
          fontFamily: 'ui-monospace, Menlo, monospace',
          fontSize: 18, letterSpacing: '0.2em', textTransform: 'uppercase',
        }}>
          {launchDate} — Upvote Today
        </div>
        <div style={{
          fontSize: 32, fontStyle: 'italic', color: '#DA552F',
        }}>
          producthunt.com
        </div>
      </div>
    </div>
  );
}
```

**Design notes.** No pills, no buttons, no avatars — just a masthead, a stacked callout, the product name as the visual centerpiece, and a tour-style date strip. The italic serif treatment for the product name is what carries the poster feel.

## 2. Feature launch — typography poster

**Use case.** "Introducing X" treated as art-directed type, not a UI announcement. Cream top, cherry-red bottom band, the feature name centered in display-weight lowercase, a rotated ghosted "new" echo nestled below it. Portrait 1200×1500.

![Feature launch poster — top section cream with massive centered lowercase "drafts." in deep ink, ghosted outlined rotated "new" echo below-left, "INTRODUCING · AUTUMN 26 · V2" monospace eyebrow at top, bottom cherry-red band with italic "A new way to design together. Available now." and oversized arrow](/examples/feature-launch.jpg)

**Path.** `.lerret/marketing/feature-launch.jsx`

```jsx
export const meta = {
  dimensions: { width: 1200, height: 1500 },
  label: 'Feature launch — typography poster',
  tags: ['poster', 'announcement', 'feature'],
  propsSchema: {
    feature: { type: 'string', default: 'Drafts', required: true },
    version: { type: 'string', default: 'V2' },
    issueDate: { type: 'string', default: 'AUTUMN 26' },
    footnote: { type: 'string', default: 'A new way to design together. Available now.' },
  },
};

export default function FeatureLaunchPoster({ feature, version, issueDate, footnote }) {
  return (
    <div style={{
      width: '100%', height: '100%', boxSizing: 'border-box',
      background: '#F0EAD9', color: '#181614',
      fontFamily: 'system-ui, -apple-system, sans-serif',
      position: 'relative', overflow: 'hidden', padding: '64px 72px',
    }}>
      {/* Cherry block */}
      <div style={{
        position: 'absolute', bottom: 0, left: 0, right: 0,
        height: '38%', background: '#C4392F',
      }} />

      {/* Eyebrow */}
      <div style={{
        position: 'relative', display: 'flex', justifyContent: 'space-between',
        fontFamily: 'ui-monospace, Menlo, monospace',
        fontSize: 13, letterSpacing: '0.3em', textTransform: 'uppercase', opacity: 0.8,
      }}>
        <span>Introducing · {issueDate}</span>
        <span>{version}</span>
      </div>

      {/* Feature word — sized to fit comfortably within the canvas */}
      <div style={{
        position: 'absolute', left: 72, right: 72, top: 280, textAlign: 'center',
      }}>
        <div style={{
          fontSize: 280, fontWeight: 900, lineHeight: 0.88,
          letterSpacing: '-0.055em', textTransform: 'lowercase',
        }}>
          {feature.toLowerCase()}.
        </div>
      </div>

      {/* Ghosted "new" echo, rotated */}
      <div style={{
        position: 'absolute', left: 60, top: 640,
        fontSize: 180, fontWeight: 900, letterSpacing: '-0.05em',
        color: 'transparent',
        WebkitTextStroke: '2px rgba(24,22,20,0.18)',
        transform: 'rotate(-6deg)', pointerEvents: 'none',
      }}>new</div>

      {/* Footnote on cherry block */}
      <div style={{
        position: 'absolute', left: 72, right: 72, bottom: 80,
        color: '#F0EAD9', display: 'flex',
        justifyContent: 'space-between', alignItems: 'flex-end', gap: 32,
      }}>
        <div style={{
          fontSize: 36, lineHeight: 1.15, fontStyle: 'italic',
          maxWidth: '72%',
        }}>{footnote}</div>
        <div style={{ fontSize: 64, fontWeight: 900 }}>→</div>
      </div>
    </div>
  );
}
```

**Design notes.** The poster move is **type-as-image** — the feature word IS the design. Sized at 280px (down from a more aggressive 360 in an earlier draft that bled past the canvas edge), the word fits cleanly with breathing room either side, while the outlined "new" echo nested below-left and rotated 6° gives the print-shop / risograph energy. The cherry-red band at the bottom is a flat color block, not a gradient — that's the screen-print signal.

## 3. Stars milestone — constellation poster

**Use case.** Open-source milestone celebrations belong in the gallery. Deep midnight blue, scattered glowing star dots, a giant italic-serif number as the only loud element, vertical "STARS · STARS · STARS" running text on the side. Square 1080×1080, made for Instagram and PH thumbnails.

![Stars milestone poster — deep navy background with scattered glowing star dots, hairline frame, "★ MILESTONE UNLOCKED · MMXXVI" monospace masthead, vertical outlined "STARS · STARS · STARS" rotated text on left, centered giant italic-serif "10,000" in cream, "STARS on GitHub" subtitle, repo handle bottom-left, italic "with thanks, everyone" bottom-right](/examples/stars-milestone.jpg)

**Path.** `.lerret/marketing/stars-milestone.jsx`

```jsx
export const meta = {
  dimensions: { width: 1080, height: 1080 },
  label: 'Stars milestone — constellation poster',
  tags: ['poster', 'milestone', 'github'],
  propsSchema: {
    count: { type: 'string', default: '10,000' },
    label: { type: 'string', default: 'STARS' },
    handle: { type: 'string', default: '@belikely-united/lerret' },
    note: { type: 'string', default: 'with thanks, everyone' },
  },
};

const STARS = [
  [8, 12, 4], [22, 22, 2], [40, 8, 3], [60, 18, 5], [78, 12, 3], [92, 22, 4],
  [12, 35, 2], [88, 38, 2], [4, 52, 3], [96, 56, 2],
  [10, 75, 5], [25, 88, 3], [42, 92, 2], [58, 82, 4], [75, 90, 3], [90, 78, 5],
  [50, 60, 2], [30, 50, 3], [70, 48, 2],
];

export default function StarsPoster({ count, label, handle, note }) {
  return (
    <div style={{
      width: '100%', height: '100%', boxSizing: 'border-box',
      background: '#0B1426', color: '#F5EBD8',
      fontFamily: 'Georgia, "Times New Roman", serif',
      position: 'relative', overflow: 'hidden', padding: '72px',
    }}>
      {STARS.map(([x, y, size], i) => (
        <div key={i} style={{
          position: 'absolute', left: `${x}%`, top: `${y}%`,
          width: size, height: size, background: '#F5EBD8', borderRadius: '50%',
          boxShadow: `0 0 ${size * 3}px rgba(245,235,216,0.8)`,
          opacity: 0.5 + (i % 5) * 0.1,
        }} />
      ))}

      {/* Hairline frame + vertical STARS text + centered number + footer */}
      <div style={{
        position: 'absolute', inset: 40,
        border: '1px solid rgba(245,235,216,0.18)', pointerEvents: 'none',
      }} />

      <div style={{
        position: 'absolute', left: 56, top: '50%',
        transform: 'translateY(-50%) rotate(-90deg)', transformOrigin: 'left center',
        fontFamily: 'system-ui, -apple-system, sans-serif',
        fontSize: 22, fontWeight: 800, letterSpacing: '0.5em',
        color: 'transparent', WebkitTextStroke: '1px rgba(245,235,216,0.55)',
      }}>{label} · {label} · {label}</div>

      <div style={{
        position: 'absolute', inset: 0,
        display: 'flex', flexDirection: 'column',
        alignItems: 'center', justifyContent: 'center', textAlign: 'center',
      }}>
        <div style={{
          fontFamily: 'system-ui, -apple-system, sans-serif',
          fontSize: 12, letterSpacing: '0.6em',
          color: 'rgba(245,235,216,0.5)', marginBottom: 18,
          textTransform: 'uppercase', fontWeight: 600,
        }}>We just crossed</div>
        <div style={{
          fontSize: 280, fontWeight: 400, letterSpacing: '-0.05em',
          lineHeight: 0.85, fontStyle: 'italic',
          fontVariantNumeric: 'tabular-nums',
        }}>{count}</div>
        <div style={{
          fontFamily: 'system-ui, -apple-system, sans-serif',
          fontSize: 18, letterSpacing: '0.5em', marginTop: 24,
          textTransform: 'uppercase', fontWeight: 800,
        }}>{label} on GitHub</div>
      </div>
    </div>
  );
}
```

**Design notes.** The italic serif number is doing the heavy lifting — it reads as poetry, not a metric. Vertical `transform: rotate(-90deg)` text with `-webkit-text-stroke` gives the side-spine pattern you see on book covers and movie posters. Dot stars with `box-shadow` glow read as actual light sources.

## Why these patterns

The posters above lean on conventions from print, not from the web. A few that translate especially well via plain React + inline styles:

- **`-webkit-text-stroke` + `color: transparent`** → outlined display type without an SVG. Survives every Chromium export pipeline.
- **`transform: rotate()` on text blocks** → editorial diagonals, rubber-stamp angles, vertical book-spine text.
- **`writing-mode: vertical-rl` or rotated transform** → side-spine text for masthead grids.
- **Solid color blocks (`background`, no gradient)** → screen-printed flat-color feel, prints well at any scale.
- **Numbered ledger rows** → editorial gridding without box-card UI patterns. Each row is a `border-bottom`; indices live in monospace.
- **Mixed serif + sans + monospace** → masthead grids almost always need all three. System default Georgia + system-ui sans + ui-monospace gets you 90% of the way without webfont overhead.

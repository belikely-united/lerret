# FAQ

This page is a stub.

## Is Lerret a Figma alternative?

No. Figma is a vector design tool with a proprietary file format. Lerret renders your existing React components as a canvas — your source of truth stays code.

## Does Lerret upload my files?

No. Hosted mode reads and writes locally via the browser's File System Access API; the static studio has no backend.

## What browsers does the hosted studio support?

Chromium browsers (Chrome, Edge, Brave, Arc) — the File System Access API requires it. The CLI works everywhere Node runs.

## Is it production-ready?

The 0.1 series is the first public release. APIs may shift. Open an issue on GitHub if anything breaks.
